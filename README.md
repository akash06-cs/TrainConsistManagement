# Train Consist Management App

## Overview
This project demonstrates how core Java data structures are applied to manage a train consist in a real-world scenario.

## Features
- Dynamic bogie management using ArrayList
- Duplicate prevention using HashSet
- Train ordering using LinkedList
- Ordered + unique formation using LinkedHashSet
- Bogie capacity mapping using HashMap

## Tech Stack
- Core Java
- Java Collections Framework
- Git & GitHub

## Project Structure
Each use case is implemented in a separate branch:
- UC1 → Initialization
- UC2 → ArrayList operations
- UC3 → HashSet (uniqueness)
- UC4 → LinkedList (ordering)
- UC5 → LinkedHashSet (order + uniqueness)
- UC6 → HashMap (mapping)

## Learning Outcome
This project bridges theoretical data structures with real-world system design.
